# Retrofit With MVVM Architecture

This project demonstrates how to use Retrofit with MVVM (Model-View-ViewModel) architecture in Android using Kotlin.

## Features

- MVVM Architecture
- Retrofit for API Calls
- ViewModel
- State Management
- Clean Code Structure
- Jetpack Compose UI

## Tech Stack

- Kotlin
- Jetpack Compose
- Retrofit
- Coroutines
- ViewModel
- Android Studio

## Project Structure

```
app
│
├── api
├── repository
├── viewmodel
├── ui
└── model
```

## How It Works

1. Retrofit fetches data from the API.
2. Repository manages data operations.
3. ViewModel provides data to the UI.
4. Jetpack Compose displays the data.

## Learning Objectives

- Understand MVVM Architecture
- Learn Retrofit Integration
- Work with Coroutines
- Manage UI State using ViewModel
- Build scalable Android applications
